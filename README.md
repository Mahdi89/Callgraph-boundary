# Callgraph boundary

A tool to analyze the call graph in `C` and determine the paths between the function calls.
It is useful for understanding complex projects.

## Requirements

- llvm >v5.0.0
- clang >v3.8.0
- llvm optimizer (opt)
- go >1.8

## Build and Install

`make build`

and 

`make run`

## Test and Benchmarking

`make test`
`make bench`
